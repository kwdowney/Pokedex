###############

Project description: database for storing entries that can be tagged with a picture.

###############

Compile instructions:

user@computer~/path/to/files $ qmake -project 
user@computer~/path/to/files $ qmake
user@computer~/path/to/files $ make

###############

Usage instructions: Text goes in "name" field to name the entry. To upload the image, [/path/to/image/image.jpg] goes in "image" text box. (e.g. /images/gengar.jpg)

###############
